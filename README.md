# PRC Civil Litigation Drafting & Evidence Skill

一个面向中国民事诉讼实务的 Codex Skill，用于协助起草、审阅和核验诉讼文书，以及整理证据目录与证据材料。

## 能力范围

- 起草、修改和定稿民事答辩状、民事起诉状、鉴定申请书；
- 制作证据目录及民事证据登记表；
- 整理、合并、添加书签、连续编页并核验证据 PDF；
- 对 DOCX、PDF、扫描件、聊天记录、支付记录及其他电子数据进行证据导向审查；
- 标记事实或页码缺口，避免将未经证实的事项写成确定事实。

## 核心设计

本 Skill 并非仅生成文本。它将诉讼请求、抗辩要件、举证责任、争议焦点与具体证据建立对应关系，并强调以下顺序：

1. 确定证据范围、分组、编号和排序；
2. 合并、校正和连续编页证据 PDF；
3. 回填证据目录、登记表和诉讼文书脚注；
4. 对名称、日期、金额、主体信息、页码、书签、签名和印章进行一致性核验。

当事实或证据不足时，Skill 要求在工作稿中明确标记 `【待确认】`，而非虚构内容。涉及现有法律 Word 文档的实质性审阅，建议使用 Track Changes 和批注，并将审阅身份配置为中性的“审阅律师”。

## 使用方式

将整个目录放入 Codex Skills 目录后，显式调用：

```text
$draft-defense-evidence 按标准流程处理本案的答辩状和证据材料。
```

也可直接描述任务，例如“按标准流程制作证据目录并合并证据 PDF”。应当始终提供本案材料，并在定稿前由有资质的法律专业人士完成审查和确认。

## 隐私与免责声明

本公开版本已移除个人姓名和机器本地路径。它仅提供工作流与文书组织规则，不构成法律意见，也不能替代律师对具体事实、证据、适用法律和程序要求的独立判断。
